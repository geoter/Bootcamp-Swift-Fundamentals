# Optional Chaining Exercises

## Exercise 1
Create a `Person` struct with an optional `Address` property. The `Address` struct should have an optional `city` property. Use optional chaining to print the city of the person's address if it exists.

### Solution
<details>
<summary>Show Solution</summary>

```swift
struct Person {
    var name: String
    var address: Address?

    struct Address {
        var city: String?
    }
}

let person = Person(name: "John", address: Person.Address(city: "New York"))

if let city = person.address?.city {
    print("\(person.name) lives in \(city)")
} else {
    print("\(person.name) does not have a city listed.")
}
```
</details>

## Exercise 2
Create a `Company` struct with an optional `CEO` property. The `CEO` struct should have an optional `email` property. Use optional chaining to print the CEO's email if it exists.

### Solution
<details>
<summary>Show Solution</summary>

```swift
struct Company {
    var name: String
    var ceo: CEO?

    struct CEO {
        var email: String?
    }
}

let company = Company(name: "TechCorp", ceo: Company.CEO(email: "ceo@techcorp.com"))

if let email = company.ceo?.email {
    print("The CEO's email is \(email)")
} else {
    print("The CEO's email is not listed.")
}
```
</details>

## Exercise 3
Create a `Library` struct with an optional `Book` property. The `Book` struct should have an optional `author` property. Use optional chaining to print the author's name if it exists.

### Solution
<details>
<summary>Show Solution</summary>

```swift
struct Library {
    var name: String
    var book: Book?

    struct Book {
        var author: String?
    }
}

let library = Library(name: "City Library", book: Library.Book(author: "J.K. Rowling"))

if let author = library.book?.author {
    print("The book's author is \(author)")
} else {
    print("The book's author is not listed.")
}
```
</details>
